# Tareas: No se puede seleccionar la función de usuario al crear una tarea

>[!NOTE]
>
>Este problema se corrigió el 21 de diciembre de 2025.

Cuando un usuario está creando una tarea e intenta asignar una función de usuario, la opción no está presente y el usuario no puede asignarla.

**Solución**

Cree la tarea y, a continuación, edite la tarea para asignar la función de usuario en Asignaciones avanzadas.

_Notificado por primera vez el 28 de noviembre de 2025._
