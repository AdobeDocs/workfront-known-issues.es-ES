# Tareas: No se puede seleccionar el rol de usuario al crear una tarea

Cuando un usuario está creando una tarea e intenta asignar una función de usuario, la opción no está presente y el usuario no puede asignar una función de usuario.

**Solución**

Cree la tarea y, a continuación, edite la tarea para asignar el rol de usuario en Asignaciones avanzadas.

_Notificado por primera vez el sábado, 28 de noviembre de 2025._
