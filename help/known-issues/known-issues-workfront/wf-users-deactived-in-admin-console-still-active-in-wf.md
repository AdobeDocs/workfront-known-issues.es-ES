# Usuarios: Los usuarios eliminados de Adobe Admin Console siguen activos en Workfront

Cuando un administrador elimina un usuario en Adobe Admin Console, no se elimina de Workfront. El usuario sigue visible y activo en Workfront.

_Notificado por primera vez el 3 de febrero de 2026._
