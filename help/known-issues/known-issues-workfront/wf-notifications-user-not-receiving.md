# Notificaciones: el usuario no recibe las notificaciones esperadas

Cuando se produce un evento que debería enviar una notificación a un usuario, ese usuario no recibe la notificación. Esto puede ocurrir incluso si la configuración de notificación del usuario permite la notificación.

Este problema afecta a las notificaciones tanto en la aplicación como por correo electrónico.

_Notificado por primera vez el 12 de febrero de 2026._
