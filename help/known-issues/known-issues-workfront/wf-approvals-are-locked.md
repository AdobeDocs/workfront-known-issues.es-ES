# Aprobaciones: las aprobaciones se bloquean automáticamente

Cuando un usuario intenta tomar una decisión sobre una prueba, la aprobación está bloqueada y el usuario no puede tomar una decisión. Esto puede ocurrir aunque no haya razón para bloquear la prueba.

_Notificado por primera vez el 16 de febrero de 2026._
