# Tareas: las ediciones en línea de una tarea no se guardan

>[!NOTE]
>
>* Este problema se corrigió el 15 de diciembre de 2025.
>* Este problema se aplica a la experiencia de tareas heredadas.

Cuando un usuario realiza cambios en línea en una tarea en la experiencia de tareas heredadas y guarda, los cambios no se guardan.

**Solución**

Para realizar algunos cambios, puede utilizar la nueva experiencia de tarea.

_Informado por primera vez el 4 de diciembre de 2025._
