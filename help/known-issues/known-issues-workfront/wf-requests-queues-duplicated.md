# Solicitudes: las colas de solicitudes aparecen varias veces

>[!NOTE]
>
>* Este problema se corrigió el 8 de enero de 2026.
>* Este problema se ha notificado en la experiencia de creación de informes heredada.

Cuando un usuario comienza a crear una solicitud y selecciona una cola de solicitudes, esta aparece varias veces en la lista desplegable.
