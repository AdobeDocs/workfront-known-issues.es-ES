# Inicio de sesión: Workfront no se carga después del inicio de sesión

Cuando un usuario inicia sesión en Workfront, Workfront no se carga. Si el usuario actualiza la página, Workfront se carga según lo esperado.

**Solución**

Actualice la página.

_Notificado por primera vez el jueves, 14 de enero de 2026._
