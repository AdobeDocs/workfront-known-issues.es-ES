# Pruebas: no se puede generar una prueba desde un archivo PowerPoint

Cuando un usuario intenta generar una prueba desde un PowerPoint, esta no se crea y el usuario ve un mensaje de error.

**Solución**

Exportar el archivo PowerPoint como PDF y, a continuación, crear una prueba desde el PDF.

_Notificado por primera vez el 21 de marzo de 2025._
