# Tareas: La lista de tareas grande salta inesperadamente a un lugar diferente de la lista

Cuando un usuario está viendo una lista de tareas grande y actualiza una tarea o expande una tarea principal, la lista de tareas salta inesperadamente a un lugar diferente de la lista.

_Notificado por primera vez el 22 de enero de 2026._
