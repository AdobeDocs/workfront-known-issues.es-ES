# Integraciones: Las notificaciones de Slack desde Workfront no se activan

Cuando se produce un evento en Workfront que debería almacenar en déclencheur una notificación de Slack, no se envía la notificación.

Se ha informado de este problema en los eventos siguientes:

* Un usuario agrega comentarios a una tarea o proyecto asignado al usuario afectado
* El usuario afectado se menciona o se etiqueta en un comentario en Workfront.

Esto puede ocurrir aunque las preferencias de notificación del usuario estén configuradas para enviar la notificación en Slack.

_Notificado por primera vez el 2 de febrero de 2026._
