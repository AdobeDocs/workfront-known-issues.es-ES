# Plantilla de diseño: la página de destino es incorrecta

>[!NOTE]
>
>Este problema se ha cerrado porque no existe ningún error.

Cuando un usuario con licencia de solicitante se asigna a una plantilla de diseño que tiene la página de solicitudes configurada como página de destino, se dirige a ese usuario a la página de inicio como página de destino.

Esto se ha notificado para las licencias de solicitante.

_Notificado por primera vez el sábado, 22 de marzo de 2024._
