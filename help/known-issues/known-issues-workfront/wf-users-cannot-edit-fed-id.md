# Usuarios: no se puede editar el identificador de federación de un usuario

>[!NOTE]
>
>* Este problema se corrigió el 23 de septiembre de 2025.
>* Este problema afecta a las organizaciones que aún no han migrado al sistema de administración de identidades de Adobe

Cuando un administrador intenta actualizar el identificador de federación de un usuario, el campo de dicha información no está disponible y el administrador no puede actualizarlo.

Esto significa que el usuario no puede iniciar sesión en Workfront.

_Notificado por primera vez el 9 de septiembre de 2025._
