# Solicitudes: los permisos no se heredan al crear una solicitud

>[!NOTE]
>
>Este problema se corrigió el 19 de diciembre de 2025.

Cuando un usuario crea una solicitud, esta no hereda los permisos. Esto significa que es posible que los usuarios no puedan acceder a la solicitud.

_Notificado por primera el 20 de noviembre de 2025._
