# Tareas: las horas planificadas no se guardan al crear la tarea

>[!NOTE]
>
>Este problema se corrigió el 3 de diciembre de 2025.

Cuando un usuario crea una tarea, introduce las horas planificadas y las guarda, estas no se guardan.

**Solución**

Cree la tarea y, a continuación, edítela para añadir las horas planificadas.

_Notificado por primera vez el 10 de octubre de 2025._
