# Tareas: Mensaje “Acceso insuficiente” al editar en línea las tareas

>[!NOTE]
>
>Este problema se corrigió el 21 de noviembre de 2025.

Cuando un usuario intenta editar en línea una tarea de una lista o informe, no puede editar la tarea y ve el mensaje “No cuenta con acceso suficiente para realizar asignaciones a esta tarea”.

Esto ocurre cuando el usuario ha heredado los permisos de la tarea y la opción Realizar asignaciones está deshabilitada mediante el uso compartido de proyectos.

**Solución**

Abra la tarea y edítela directamente.

_Notificado por primera vez el 10 de noviembre de 2025._
