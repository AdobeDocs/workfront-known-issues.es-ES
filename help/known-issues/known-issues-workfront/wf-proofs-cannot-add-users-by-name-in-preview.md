# Pruebas: no se pueden añadir usuarios a una prueba por su nombre en el entorno de vista previa

>[!NOTE]
>
>Se ha informado de este problema en el entorno de zona protegida de vista previa.

Cuando un usuario intenta agregar otro usuario a una prueba y busca a dicho usuario por su nombre, el nombre no aparece y el usuario no puede seleccionarlo para añadir al otro usuario a la prueba.

Esto puede ocurrir incluso después de actualizar el entorno de vista previa de espacio aislado.

_Informado por primera vez el 5 de mayo de 2025._
