# Asignaciones: No se puede guardar el cambio de asignación avanzado de un informe

Cuando un usuario intenta realizar un cambio de asignación avanzada desde un informe, el cambio no se guarda y el usuario ve el siguiente mensaje:

&quot;422: APIModel INTERNAL no admite el campo removeOldAssignments&quot;

**Solución**

Puede cambiar las asignaciones avanzadas desde la lista de tareas del proyecto.

_Notificado por primera vez el 4 de febrero de 2026._
