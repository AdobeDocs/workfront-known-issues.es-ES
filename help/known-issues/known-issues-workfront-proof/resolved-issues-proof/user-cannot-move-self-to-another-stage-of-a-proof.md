# Pruebas: El usuario no puede pasar a otro paso de una prueba

>[!NOTE]
>
>Este problema se corrigió el 24 de junio de 2022.

Cuando un usuario está viendo el flujo de trabajo de prueba de una prueba e intenta arrastrarse a un paso diferente de la prueba, el nombre del usuario vuelve al escenario original y no se añade al escenario deseado.

Informe por primera vez el 2 de junio de 2022.
